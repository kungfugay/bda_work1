### 3. 세부 분석 주제 및 질문

1. 지역별 월별 평균 미세먼지 농도 분석
2. 미세먼지 농도가 가장 높은 지역과 낮은 지역 분석
3. 도시대기와 관련하여 ‘점 오염원’과 ‘이동 오염원’에 따른 대기질 분석
